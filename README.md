# Cathy.js
Simple chatbot package based on simple ML.

# Installing

```sh
npm install cathy.js
```

# Features
- No api key required
- fast
- simple
- easy
- Machine Learning

> # It wont save messages, you have to save it on your own :)

# Getting Started

```js
const Cathy = require("cathy.js");
const cathy = new Cathy();
```

# Chatting

```js
const message = cathy.chat("hello");
console.log(message);
```
